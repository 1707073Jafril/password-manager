## Password Manager : A C++ based simple password manager using OOP, Database & CRUD operation, Hashing

# Features  
- Authentication(Login, Register, Logout)
- Security (Hashing)
- Store Password in Database and CRUD operation

# Technologies 
Langugae : C++
Database : PostgreSQL
